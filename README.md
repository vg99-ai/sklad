# Sklad App (MVP)

Простое офлайн Android-приложение для учёта движения товаров между складом, магазинами и покупателями.

## Что уже реализовано (MVP)
- Kotlin + Jetpack Compose + MVVM.
- Room база данных полностью локально на устройстве.
- Сущности: Product, Location, Stock, Incoming, Movement, Sale, WriteOff, OperationLog.
- Основной сценарий движения остатков через транзакции:
  - приход;
  - перемещение;
  - продажа;
  - запись в журнал операций.
- Главные экраны: Товары, Остатки, Приход, Перемещение, Продажа, Журнал, Отчёты.
- Заглушка экрана сканера штрихкода (под будущую интеграцию ML Kit).
- Тестовые данные первого запуска: 2 товара, склад + 3 магазина, стартовые остатки.

## Как открыть проект
1. Откройте Android Studio.
2. Выберите **Open** и укажите папку проекта `sklad`.
3. Дождитесь Gradle Sync.

## Как запустить
1. Подключите Android-устройство или запустите эмулятор.
2. Нажмите **Run app**.
